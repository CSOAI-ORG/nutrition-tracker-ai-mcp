# Nutrition Tracker AI MCP

Track daily food intake, sum macros and calories, and compare against personalized targets.

## Tools

- `log_food` — Log a food item with macros.
- `get_daily_summary` — Return total macros and calories for the day.
- `compare_to_targets` — Compare daily totals against personalized targets.

## Install

```bash
pip install -e .
```

## Run

```bash
python -m nutrition-tracker-ai-mcp.server
```
